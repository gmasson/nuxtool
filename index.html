Criado a partir de uma necessidade pessoal, o NuxTool vem com ferramentas e atalhos úteis para o uso cotidiano no ambiente Linux (com distribuições Debian ou derivados).

Modular

Desenvolve seus módulos ou instale módulos de terceiros

Suporte multilinguagens

Defina seu idioma de preferência conforme o suporte de cada módulo

Fácil de utilizar

Todos os atalhos estão disponiveis através do comando help

Open Source

Mantido pela comunidade sob a licença MIT

Como Usar

Primeiramente faça o download do programa, cujo também pode ser feito com o seguinte comando no terminal:
git clone https://github.com/gmasson/nuxtool.git
em seguida, dê permissão ao arquivo para que possa executa-lo corretamente
chmod -R 750 nuxtool
agora abra o arquivo, se preferir, abra-o pelo terminal usando o comando:
./nuxtool/nuxtool.sh

Documentação

Todas os comandos disponiveis do projeto você encontra dentro do software, execute o programa no terminal de sua preferência e digite o comando help.
