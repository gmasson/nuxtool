Ferramentas para GNU / Linux baseados em Debian

Criado a partir de uma necessidade pessoal, o NuxTool vem com ferramentas e atalhos úteis para o uso cotidiano no ambiente Linux (com distribuições Debian ou derivados).

Modular

Desenvolva seus módulos ou instale módulos de terceiros

Suporte multilinguagens

Defina seu idioma de preferência conforme o suporte de cada módulo

Fácil de utilizar

Todos os atalhos estão disponiveis através do comando help

Open Source

Mantido com carinho pela comunidade sob a licença MIT

Como Usar
Primeiramente faça o download do programa, cujo também pode ser feito com o seguinte comando no terminal:
git clone https://github.com/gmasson/nuxtool.git
em seguida, dê permissão ao arquivo para que possa executa-lo corretamente
chmod -R 750 nuxtool
agora abra o arquivo, se preferir, abra-o pelo terminal usando o comando:
./nuxtool/nuxtool.sh
Documentação
Todas os comandos disponiveis no projeto você encontra dentro do software, execute o programa no terminal de sua preferência e digite o comando help.

Alterando o idioma padrão

Por padrão o Nuxtool está habilitado em Inglês, para alterar, abra o arquivo config.sh no editor de sua preferência e altere o trecho language="en" para a sigla do idioma que deseja, exemplo: language="pt-br"
